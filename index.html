What is an Input Device?

An Input Device is a peice of hardware used to select the data in the computer. Look below at some examples and what they do.

Examples of Input devices:

1) Mouse.

A mouse is an input device, as it is used to click and select an item which you want to see the result of.

2) Keyboard.

A Keyboard is an input device, as it is used to insert the letters or text into the computer to process.

3) Microphone.

A Microphone is an Input device, as it takes in your sound, and then processes it.
